Qual a diferença entre Section e Div?

Section

A tag section carrega consigo a responsabilidade semântica do HTML5, ela foi desenvolvida para representar seções de um documento, essa é a idéia principal.

Isso quer dizer que a section tem um propósito, ela não deve ser usada apenas por necessidades de estilização.

Div

O elemento div é apropriado para partes da página que você deseja dividir para fins diferentes da semântica.

É utilizado para estilizar layout de containers e para faze divisões em blocos no HTML facilitando a leitura do código.
